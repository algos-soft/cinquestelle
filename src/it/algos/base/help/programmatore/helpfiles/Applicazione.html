Applicazione


• Un' applicazione e' il modulo iniziale di un programma (quello che parte per primo)

• Un' applicazione viene sempre fatta partire dall'utente:
-- doppio click sull'icona dell'applicazione
-- viene invocata la JVM (Java Virtual Machin)
-- vengono caricate le classi, ....
-- viene lanciato il metodo main

• La classe iniziale di un' applicazione deve essere xxxPartenza (una sottoclasse di Partenza)
