Array di finestre lista
Set di finestre di tipo lista mantenuto dal modulo

 

• Ogni modulo ha un array di finestre lista differenti tra loro
• Viene mantenuto un puntatore alla finestra corrente
• L'array e' di tipo ArrayList; nella fase di inizializzazione del Modulo, viene creata una finestra lista di default ed aggiunta all'array nella prima posizione
• Dalle classi xxxModulo specifiche, si possono costruire finestre lista diverse in tre modi:
-- sostituendole alla prima finestra (quella di default), che viene quindi scaricata
-- sostituendole ad una qualsiasi altra finestra, che viene quindi scaricata
-- aggiungendole in coda all'array
