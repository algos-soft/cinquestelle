StrutturaModulo
Abstract Data Types per la struttura dei dati di un modulo

• Serve come contenitore di tutte le informazioni necessarie al funzionamento di un modulo; riferimenti (puntatori) a tutte le classi coinvolte nella gestione del modulo

• Tipicamente viene costruita un'instanza di StrutturaModulo nella classe astratta AzioneAvviaModulo e vengono regolati alle classi default tutti i puntatori; poi si invoca il metodo avviaModulo della classe (col parametro obbligatorio di inizio codificato) ed a questo punto ci sono tutte le informazioni che servono e viene invocato il metodo avviaGestore della classe gestore
