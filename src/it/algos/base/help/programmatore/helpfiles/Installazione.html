4 - Installazione

a) questo programma opera sotto il controllo della JVM (Java Virtual Machine), se questa non e' gia' installata, occorre provvedere ad installarla prima di poter utilizzare il programma.

b) questo programma memorizza i propri dati in una base dati SQL. Occorre fornire, installare, configurare ed avviare un database prima di poter operare sul programma. Puo' essere utilizzato qualunque motore sql: MySql, Postgres, Oracle, Informix, ecc.
