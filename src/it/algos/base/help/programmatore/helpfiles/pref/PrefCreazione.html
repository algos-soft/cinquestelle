Creazione delle preferenze

aprile 2007

La classe principale di preferenze Pref, contiene le enumeration del progetto di base e viene creata automaticamente dalla classe Progetto

Per utilizzare preferenze specifiche di un progetto, occorre scrivere una classe XxxxPref, sotoclasse di Pref

Quando si crea una sottoclasse di Pref, occorre copiare una enumeration completa da Pref e modificarla come necessario

Nel metodo .inizia() della classe XxxxModulo, occorre creare un'istanza della classe XxxxPref. Non occorre conservare un riferimento all'oggetto cretao, perché tutta la funzionalità delle preferenze viene gestita a livello statico

Nella sottoclasse XxxxPref, si possono implementare n enumeration.

Nel metodo .inizia() della sottoclasse XxxxPref, occorre invocare super.addGruppo(Yyy.class) per ogni enumeration creata.

Sempre nel metodo .inizia() della sottoclasse XxxxPref, occorre invocare Pref.carica() dopo aver aggiunto tutte le enumeration

Il metodo Pref.carica() legge le eventuali preferenze non standard registrate sul file locale di testo (o su altro supporto permanente di memoria)
