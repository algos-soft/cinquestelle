Distribuzione


Struttura di file e cartelle di distribuzione:
- uno o più file .jar per le applicazioni eseguibili
- un eventuale file "Leggimi.txt"
- una cartella "Support" contenente:
- un file "startup.properties" di tipo Properties che contiene:
- l'indirizzo della cartella "Resources"
- la cartella "Library" che contiene una o più cartelle (una per ogni libreria). Ognuna di queste cartelle contiene un file .jar e una cartella "Support"
- una cartella "Resources" che si può trovare ovunque, che contiene:
- il file "text.jar" (i testi fissi specifici dell'applicazione)
- il file "preferences.properties" (le preferenze specifiche dell'applicazione, che possono essere relative a uno o più utenti)
- il file "images.jar" (le immagini, bottoni, icone, fissi specifici dell'applicazione)

Esempio:
[ ] = cartella

[ ]Inventario
|
+-- inventario.jar (app. specifica eseguibile)
+-- leggimi.txt (informativa)
+-- [ ]Support
|
+-- startup.properties (contiene indirizzo di [ ]Resources)
|
+-- [ ]Library
|
+-- [ ]Base
| |
| +-- base.jar
| +-- [ ]Support
|
+-- [ ]Anagrafica
|
+-- anagrafica.jar
+-- [ ]Support[ ]Resources (risorse della appplicazione specifica, può essere dovunque, tipicamente nella cartella "Support")
|
+- text.jar (testi fissi)
+- preferences.properties (preferenze)
+- images.jar
(da riprendere quando si faranno le prove di distribuzione)
